# BGT naar CO2 Monitor Script

In deze repository staan de bestanden voor het script dat BGT-gegevens (Basisregistratie Grootschalige Topografie) verwerkt voor gebruik in de CO2-monitor.

## Over dit project

Dit script helpt bij het omzetten en voorbereiden van geografische data uit de BGT zodat deze gebruikt kan worden in analyses en monitoring van CO2-gerelateerde informatie.

## Gebruik

Bekijk de bestanden in deze repository voor de scripts en configuraties die nodig zijn om het proces uit te voeren. Afhankelijk van de toepassing kan het nodig zijn om paden of instellingen aan te passen.

Dit project is bedoeld als hulpmiddel bij het verwerken van BGT-data voor toepassingen rond CO2-monitoring.
